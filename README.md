## 프로젝트 소개
이 프로젝트는 웹 기반 게임 퀴즈 인터페이스로, 다양한 게임 이미지를 보여주고 제한 시간 내에 정답을 맞추는 방식으로 진행됩니다.  
라운드 진행, 점수 계산, 타이머, 진행바 등 기본적인 게임 로직이 구현되어 있습니다.

---

## 실행 방법
1. VSCode에서 프로젝트 폴더를 엽니다.
2. `index.html` 파일을 열고, **Live Server** 확장 프로그램을 사용해 실행합니다.
   - HTML 파일을 우클릭 → **"Open with Live Server"** 선택
3. 브라우저가 자동으로 열리며 게임 화면이 표시됩니다.

또는

https://kimdeukgap.github.io/ossproject/ossproject/ 해당 주소로 접속 가능

---

## 주요 기능
- 라운드별 문제 출제 및 진행
- 제한 시간 타이머
- 정답 입력 및 즉시 피드백
- 점수 계산 및 최종 결과 표시
- 진행 상황을 보여주는 프로그레스 바

---

## 기술 스택
- **Frontend**: HTML, CSS, JavaScript  
- **개발 환경**: VSCode + Live Server
